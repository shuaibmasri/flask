# Developing AI Applications with Python and Flask – Labs

## 📌 Overview

This repository contains my hands-on lab implementations from the course **"Developing AI Applications with Python and Flask"**. The goal of these labs is to build practical experience in integrating Artificial Intelligence models into web applications using Python and Flask.

All projects were developed and tested locally, then version-controlled and published on GitHub for learning, reference, and portfolio purposes.

---

## 🎯 Objectives

* Understand how to build web applications using Flask
* Integrate AI/ML models into backend services
* Handle HTTP requests and responses in a RESTful architecture
* Work with APIs and data processing pipelines
* Deploy and structure scalable Python applications

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Create a virtual environment:

```bash
python -m venv venv
```

3. Activate the environment:

* Windows:

```bash
venv\Scripts\activate
```

* Linux/Mac:

```bash
source venv/bin/activate
```

4. Install dependencies:

```bash
pip install -r requirements.txt
```

5. Run the Flask app:

```bash
python app.py
```

6. Open your browser:

```
http://127.0.0.1:5000/
```
